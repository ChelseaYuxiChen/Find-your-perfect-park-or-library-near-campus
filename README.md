# Find Your Perfect Park or Library Near Campus

This project provides a data-driven dashboard to help users discover parks and libraries near a specified campus location. By analyzing various factors such as distance, amenities, and user preferences, the dashboard offers personalized recommendations to enhance the campus experience.

## Features

- Interactive Dashboard
- Customizable Filters
- Data Analysis

## Installation

1. **Clone the Repository**:

   ```bash
   git clone https://github.com/ChelseaYuxiChen/Find-your-perfect-park-or-library-near-campus.git

2. **Navigate to the Project Directory:**:
   ```bash
   cd Find-your-perfect-park-or-library-near-campus

3. **Set Up a Virtual Environment (Optional but recommended):**:
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # On Windows, use 'venv\Scripts\activate'

4. **Install Required Packages:**:
   ```bash
   pip install -r requirements.txt

## Usage
1. **Run the Dashboard Application:**:
   ```bash
   python data_dashboard.py

3. **Access the Dashboard:**:
   Open a web browser and navigate to the local server address displayed in the terminal (e.g., http://127.0.0.1:8050/).

3. **Interact with the Dashboard:**:
   - Input the campus location.​
   - Adjust filters and preferences.​
   - View and analyze the recommended parks and libraries.​
